序列： 在python当中， 序列就是一次按照顺序排列的值【数据集合】
在python中， 存在三种内置的序列类型：
字符串、列表、元组
优点：可以支持索引和切片的操作
特征：第一个正索引为0，指向的是左端，第一个索引为负数的时候，指向方向为右端
切片：【高级特性】可以根据下标来获取序列对象的任意[部分]数据
切片使用语法： [起始下标:结束下标:步长]==[start:end:step] step默认为1
# 下标从0开始，下标会越界，切片不会

list:python 当中非常重要的数据结构，是一种有序的数据集合
特点:
1. 支持增删改查
2. 列表中的数据是可以变化的,【数据项可以改变，内存地址不会改变】
3. 用[]来表示列表类型，数据项之间可以用逗号来分割，注意：数据项可以是任何类型的数据
4. 支持索引和切片进行操作

元组： 是一种不可变的序列，在创建之后不能做任何的修改
1. 不可变
2. 用（）创建元组类型，数据项用逗号分割
3. 元组可以是任何的类型
4. 当元组中只有一个元素时，要加上逗号，不然解释器会当作整型来处理 # tupleA = (1,) 
5. 同样可是支持切片操作
