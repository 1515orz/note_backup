# 流程控制_case

    case语句和if...elif...else语句一样都是多分支条件语句, 不过和if多分支条件语句不同的是, case语句只能判断一种条件关系, 而if语句可以判断多种条件关系
    
## case格式

```
case $变量 in
    "值1")
        main1
        ;;
    "值2")
        main2
        ;;
     *)  # 相当于else
        main3
        ;;
esac
```
> 分号是不能省略的,相当于语句结束
> 不要忘记结束语`esac`
> 表示`else`的`*`不需要加引号
