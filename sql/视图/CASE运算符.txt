# CASE运算符

```mysql
CASE
	WHEN...result
	WHEN...result
	WHEN.......
	ELSE
END AS [name]
```


类似于多重循环结构

```MySQL

SELECT 
	concat(first_name,' ',last_name) AS customers,
    points,
    CASE 
	WHEN points > 3000 THEN 'Gold'
       	WHEN points >= 2000 THEN 'Sliver'  # 当第一个条件false的时候执行第二行，所以不需要小于3000，只需要大于2000就能表达介于2000到3000
       	ELSE 'Bronze'
    END AS category
FROM customers
ORDER BY points DESC
```
