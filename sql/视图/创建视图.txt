# 创建一个视图

`CREATE VIEW [name] AS SELECT ......`


```MySQL
CREATE VIEW sales_by_clent AS 
SELECT 
    c.client_id,
    c.name,
    SUM(invoice_total) AS total_sales
FROM clients c
JOIN invoices i USING(client_id)
GROUP BY client_id, name 
```

>`SHOW CRETE VIEW name`
> 查看视图的定义

    视图功能非常强大，可以让我们不用反复地写相同的查询，并且可以随时调用，当作正常表格一样使用，大大简化我们的查询

    作用相似于一张虚拟表，但是不储存数据 ，我们的数据实际上还是储存在表中，只是把视图提供给了基础表
    
临时表模式
```
CREATE ALGORITHM=TEMPTABLE VIEW v_teachers AS 
SELECT * FROM teachers WHERE country = 'CN'
```
